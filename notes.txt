14. Useful npm scripts
Add these to package.json:

{
  "scripts": {
    "test": "playwright test",
    "headed": "playwright test --headed",
    "debug": "playwright test --debug",
    "report": "playwright show-report",
    "codegen": "playwright codegen"
  }
}

Then you can run:

npm test
npm run headed
npm run debug
npm run report
npm run codegen
