14. Useful npm scripts
Add these to package.json:

{
  "scripts": {
    "test": "playwright test",
    "headed": "playwright test --headed",
    "debug": "playwright test --debug",
    "report": "playwright show-report",
    "codegen": "playwright codegen"
  }
}

Then you can run:

npm test
npm run headed
npm run debug
npm run report
npm run codegen

A .env file stores configuration values outside your code.Your code reads these values at runtime.

await page.goto(process.env.BASE_URL!);
The ! tells TypeScript:

"I know this value exists."


Never commit secrets to Git.

.gitignore

Add:

node_modules/
playwright-report/
test-results/
.env
